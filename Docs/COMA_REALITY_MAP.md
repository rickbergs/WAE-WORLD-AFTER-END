# WAE: Coma Reality Map

Este documento mapeia como acontecimentos reais durante o coma podem aparecer de forma distorcida dentro do mundo pós-apocalíptico do jogo.

| Realidade | Interpretação no coma |
|---|---|
| Médicos conversando | Sussurros atrás das paredes |
| Familiar chamando o protagonista | Rádio falando o nome dele |
| Monitor cardíaco | Sirene distante ou bip no ambiente |
| Soro pingando | Chuva constante |
| Luz de exame | Clarão branco no céu ou lanterna impossível |
| Enfermeiros no quarto | Pessoas sem rosto no corredor |
| Procedimento médico doloroso | Ataque de criatura |
| Respiração assistida | Monstro com som de respiração mecânica |
| Sedação | Névoa, lentidão e distorção visual |
| Quarto hospitalar | Hospital abandonado dentro do jogo |
| Música colocada para o paciente | Rádio, vinil ou fita cassete no mundo |
| Alarme hospitalar | Alerta de emergência no apocalipse |
| Dor no peito | Inimigo que aperta ou imobiliza o protagonista |
| Tentativa de acordá-lo | Vozes dizendo para ele sair, correr ou acordar |
| Trauma do acidente | Estrada quebrada, carros batidos e bloqueios |
| Desejo de voltar para casa | Jornada até a chácara |
| Medo de morrer | O Último Eu |
| Vontade de permanecer inconsciente | Mundo da chácara ficando calmo demais |

## Regra de design

Cada delírio importante deve ter uma possível origem real.

Isso torna o jogo reinterpretável em uma segunda gameplay.

O jogador deve conseguir revisitar cenas antigas e perceber que o terror psicológico não era aleatório.
