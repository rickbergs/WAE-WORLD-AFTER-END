# WAE Game Systems

## Sistemas principais

- Movimento em primeira pessoa
- Interação com objetos
- Inventário
- Fome
- Sede
- Sono
- Vida
- Sanidade
- Rádio
- Música dinâmica
- Aparições psicológicas
- Eventos narrativos
- Exploração
- Coleta
- Base/abrigo

## Sanidade

A sanidade controla o quanto a realidade se mantém estável.

### Sanidade alta

- Mundo normal
- Sons realistas
- Música limpa

### Sanidade média

- Vultos rápidos
- Sons falsos
- Rádio com chiado estranho

### Sanidade baixa

- Vozes
- Objetos mudando
- Música distorcida
- Falsos sobreviventes

### Sanidade crítica

- A Multidão
- Versões do protagonista
- Mapas mentirosos
- Eventos psicológicos extremos
