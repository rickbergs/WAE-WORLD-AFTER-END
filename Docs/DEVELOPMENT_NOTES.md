# WAE Development Notes

## Objetivo imediato

Criar a demo vertical **WAE: First Silence**.

## Prioridades

1. Player em primeira pessoa.
2. Lanterna.
3. Sistema de interação.
4. Inventário simples.
5. Fome, sede, sono e sanidade.
6. Primeira rua abandonada.
7. Rádio funcional.
8. Primeiro vulto psicológico.
9. Música distorcida conforme sanidade.
10. Objetivo narrativo: encontrar outra voz.

## Regra principal

Não começar pelo mundo inteiro.

Começar por uma rua extremamente bem feita.
